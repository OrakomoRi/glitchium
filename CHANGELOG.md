## [1.0.0] - 2025-11-01

### :tada: Initial Release

The first public release of glitchium - a lightweight, high-performance library for fully randomized glitch effects.

### :sparkles: Features

#### Core Functionality
- **Glitch Effect System** - Fully randomized slice-based glitch effects with GPU acceleration
- **Multiple Play Modes** - Support for `always`, `hover`, `click`, and `manual` trigger modes
- **Layer System** - Configurable number of glitch layers for depth and intensity control
- **Shake Effect** - Optional base layer shake with adjustable intensity
- **Color Filters** - Dynamic hue rotation, saturation, and inversion effects

#### Advanced Features
- **Smooth Transitions** - Optional interpolation system for cinematic glitch effects
- **Custom Triggers** - Ability to trigger glitch on different elements than the target
- **SEO Optimization** - Built-in crawler detection to prevent indexing glitch effects
- **FPS Control** - Configurable frame rate for performance tuning
- **Manual Control** - Full programmatic control via start/stop/destroy methods

#### Performance
- :zap: GPU-accelerated transforms using `translate3d`
- :rocket: requestAnimationFrame for smooth 60fps
- :feather: Under 3KB minified and gzipped
- :package: Zero dependencies

### :globe_with_meridians: Showcase Website

Includes a full-featured SPA showcase website with:
- **SPA Architecture** - Client-side routing with History API
- **Syntax Highlighting** - highlight.js integration for code examples
- **Customization Playground** - Full-screen playground with all controls
- **Complete API Documentation** - Detailed descriptions and examples
